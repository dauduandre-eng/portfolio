About

Hi, I'm Daudu Mezenobe Andrew.

Every personal build is an asset — that's a principle I genuinely try to live by, not just a line on a website.

I started out writing my first programs in C++, learning object-oriented programming the slow, hands-on way: Code::Blocks, a lot of trial and error, and the quiet satisfaction of finally getting a class hierarchy to compile. I think that's where my habit of building things just to understand them came from, and it's stuck with me ever since.

Since 2024, I've been building real systems for real problems — a multi-vendor WhatsApp commerce platform with live Paystack billing, a fintech investment simulator running on its own custom domain, a captive-portal Wi-Fi access system for shared workspaces. In December 2025, I made the decision to focus on this full-time. It didn't feel like a leap into something new, more like finally giving my full attention to the part of the work I'd already loved.

These days I work mostly in Django and Python, with PostgreSQL handling the data underneath, and I'm currently exploring agentic AI integration with Claude's API on one of my existing projects. I still approach new problems the same way I did with that first C++ class hierarchy — build it, get curious about why it works, then go build the next thing.

Build. Learn. Share. Grow.

How I work

Tests aren't optional. Code that isn't proven to work is a draft, not a deliverable.

Simple, boring solutions beat clever ones — the fewest moving parts that solve the actual problem.

Production is the only environment that counts. Decisions get reviewed against what happens when real users hit them.
